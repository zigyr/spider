## 一、字符类（Character Classes）

### 1.1 预定义字符类

| 符号 | 含义 | 等价写法 |
|------|------|----------|
| `\d` | 数字 | `[0-9]` |
| `\D` | 非数字 | `[^0-9]` |
| `\w` | 字母、数字、下划线 | `[A-Za-z0-9_]` |
| `\W` | 非 `\w` 字符 | `[^A-Za-z0-9_]` |
| `\s` | 空白字符（空格、`\t`、`\n`、`\r`、`\f`、`\v`） | `[ \t\r\n\f\v]` |
| `\S` | 非空白字符 | `[^ \t\r\n\f\v]` |
| `.` | 除换行符 `\n` 外的任意字符 | — |

**最小 Demo：**

```python
import re

text = "订单号：A2024 金额￥99.5，电话13800138000"

# \d — 匹配所有数字
print(re.findall(r'\d', text))          # ['2', '0', '2', '4', '9', '9', '5', '1', '3', '8', ...]

# \D — 匹配所有非数字
print(re.findall(r'\D', text))          # ['订', '单', '号', '：', 'A', ' ', '金', '额', '￥', ...]

# \w — 匹配单词字符
print(re.findall(r'\w+', text))         # ['订单号', 'A2024', '金额', '99', '5', '电话13800138000']

# \s — 匹配空白
print(re.findall(r'\S+', text))         # ['订单号：A2024', '金额￥99.5，电话13800138000']

# . — 匹配任意字符（配合量词）
print(re.findall(r'￥.', text))         # ['￥9']
```

### 1.2 自定义字符类

| 写法 | 含义 |
|------|------|
| `[abc]` | 匹配 a、b、c 中**任意一个** |
| `[^abc]` | 匹配**除了** a、b、c 之外的任意字符 |
| `[a-z]` | 匹配 a 到 z 任意一个小写字母 |
| `[0-9A-Fa-f]` | 匹配十六进制字符 |
| `[一-龥]` | 匹配中文字符（常用） |

**最小 Demo：**

```python
import re

# [abc] — 字符组，匹配其中任意一个
print(re.findall(r'[Pp]ython', 'Python python pYTHON'))  # ['Python', 'python']

# [^abc] — 排除型字符组
print(re.findall(r'[^a-z]', 'Hello123'))                  # ['H', '1', '2', '3']

# [a-z] — 范围
print(re.findall(r'[A-Z][a-z]+', 'Hello World'))          # ['Hello', 'World']

# 匹配中文
print(re.findall(r'[一-龥]+', '你好World中国'))    # ['你好', '中国']
```

## 二、量词（Quantifiers）

| 写法 | 含义 | 贪婪？ |
|------|------|--------|
| `*` | 重复 0 次或多次 | 贪婪 |
| `+` | 重复 1 次或多次 | 贪婪 |
| `?` | 重复 0 次或 1 次 | 贪婪 |
| `{n}` | 恰好 n 次 | — |
| `{n,}` | 至少 n 次 | 贪婪 |
| `{n,m}` | n 到 m 次 | 贪婪 |
| `*?` `+?` `??` `{n,m}?` | 同上，**非贪婪（懒惰）** | 否 |

**最小 Demo：**

```python
import re

text = "a ab abb abbb"

# * — 0次或多次
print(re.findall(r'ab*', text))          # ['a', 'ab', 'abb', 'abbb']

# + — 1次或多次
print(re.findall(r'ab+', text))          # ['ab', 'abb', 'abbb']

# ? — 0次或1次
print(re.findall(r'ab?', text))          # ['a', 'ab', 'ab', 'ab']

# {n,m} — 精确控制
print(re.findall(r'ab{2,3}', text))      # ['abb', 'abbb']

# 贪婪 vs 非贪婪
html = "<div>内容1</div><div>内容2</div>"
print(re.findall(r'<div>.*</div>', html))   # 贪婪：['<div>内容1</div><div>内容2</div>']
print(re.findall(r'<div>.*?</div>', html))  # 非贪婪：['<div>内容1</div>', '<div>内容2</div>']
```

## 三、锚点（Anchors / 位置修饰符）

| 符号 | 含义 |
|------|------|
| `^` | 字符串开头（`re.M` 下也匹配行首） |
| `$` | 字符串结尾（`re.M` 下也匹配行尾） |
| `\b` | 单词边界 |
| `\B` | 非单词边界 |
| `\A` | 仅字符串开头（不受 `re.M` 影响） |
| `\Z` | 仅字符串结尾（不受 `re.M` 影响） |

**最小 Demo：**

```python
import re

# ^ / $ — 行首行尾
print(re.findall(r'^\d+', '123abc\n456def'))          # ['123']
print(re.findall(r'\w+$', 'abc 123'))                 # ['123']

# re.M 多行模式
print(re.findall(r'^\d+', '123abc\n456def', re.M))    # ['123', '456']

# \b — 单词边界
print(re.findall(r'\bpy\b', 'py file, python copy'))  # ['py']（只匹配独立的 py）

# \B — 非单词边界
print(re.findall(r'py\B', 'python pylon py'))         # ['py', 'py']（py 后面还有字母的）
```

## 四、分组与或（Groups & Alternation）

| 写法 | 含义 |
|------|------|
| `(...)` | 捕获组，可提取子串 |
| `(?:...)` | 非捕获组，只分组不提取 |
| `(?P<name>...)` | 命名捕获组 |
| `(?P=name)` | 引用命名组内容 |
| `\1` `\2` | 反向引用第 n 个捕获组 |
| `a\|b` | 或，匹配 a 或 b |

**最小 Demo：**

```python
import re

# ( ) 捕获组 — 提取子内容
m = re.search(r'(\d{3})-(\d{4})', '电话: 138-0000')
print(m.group(0))    # '138-0000'  全部匹配
print(m.group(1))    # '138'       第一个捕获组
print(m.group(2))    # '0000'      第二个捕获组

# (?: ) 非捕获组 — 只分组不保存
print(re.findall(r'(?:http|https)://', 'https://a.com http://b.com'))
# ['https://', 'http://']

# (?P<name> ) 命名组
m = re.search(r'(?P<area>\d{3})-(?P<num>\d{4})', '电话: 138-0000')
print(m.group('area'))  # '138'
print(m.group('num'))   # '0000'

# | 或
print(re.findall(r'jpg|png|gif', 'a.jpg b.png c.gif d.bmp'))
# ['jpg', 'png', 'gif']
```

## 五、标志位（Flags / 编译修饰符）

| 标志 | 简写 | 含义 |
|------|------|------|
| `re.IGNORECASE` | `re.I` | 忽略大小写 |
| `re.MULTILINE` | `re.M` | `^` `$` 匹配每行的首尾 |
| `re.DOTALL` | `re.S` | `.` 也匹配换行符 `\n` |
| `re.VERBOSE` | `re.X` | 允许正则内写注释，忽略空白 |
| `re.ASCII` | `re.A` | `\w` `\d` `\s` 等仅匹配 ASCII |

**最小 Demo：**

```python
import re

# re.I — 忽略大小写
print(re.findall(r'python', 'Python PYTHON python'))       # ['python']
print(re.findall(r'python', 'Python PYTHON python', re.I)) # ['Python', 'PYTHON', 'python']

# re.S — . 匹配换行符
text = "第一行\n第二行"
print(re.findall(r'第.*行', text))          # ['第一行']（. 不匹配 \n）
print(re.findall(r'第.*行', text, re.S))    # ['第一行\n第二行']

# re.X — 宽松模式，可写注释
pattern = r"""
    \d{3}   # 区号，三位数字
    -       # 分隔线
    \d{4}   # 号码，四位数字
"""
print(re.search(pattern, '电话: 138-0000', re.X).group())  # '138-0000'

# re.A — ASCII 限定
print(re.findall(r'\w+', '你好hello', re.A))  # ['hello']（不含中文）

# 组合使用：|
print(re.findall(r'python', 'Python PYTHON python', re.I | re.A))
```

## 六、re 模块核心函数

| 函数 | 用途 | 返回值 |
|------|------|--------|
| `re.match(pattern, string)` | 从**开头**匹配 | Match 对象或 None |
| `re.search(pattern, string)` | 搜索**第一个**匹配 | Match 对象或 None |
| `re.findall(pattern, string)` | 返回**所有**匹配 | list |
| `re.finditer(pattern, string)` | 返回**迭代器** | iterator of Match |
| `re.sub(pattern, repl, string)` | 查找替换 | str |
| `re.split(pattern, string)` | 按模式分割 | list |
| `re.compile(pattern, flags)` | 预编译（复用提升性能） | Pattern 对象 |

**最小 Demo：**

```python
import re

text = "Python 3.12 发布，Python 3.11 停更。网址 https://python.org"

# re.match — 从开头匹配（开头不匹配则返回 None）
m = re.match(r'Python', text)
print(m.group() if m else '未匹配')  # 'Python'

# re.search — 搜索第一个
m = re.search(r'\d\.\d+', text)
print(m.group())                     # '3.12'

# re.findall — 所有匹配
print(re.findall(r'Python', text))   # ['Python', 'Python']

# re.finditer — 迭代器（适合大文本）
for m in re.finditer(r'Python', text):
    print(m.start(), m.end())        # 0 6 / 16 22

# re.sub — 替换
print(re.sub(r'Python', '🐍', text))
# '🐍 3.12 发布，🐍 3.11 停更。网址 https://python.org'

# re.split — 分割
print(re.split(r'[,，]', '苹果,香蕉，橘子'))  # ['苹果', '香蕉', '橘子']

# re.compile — 预编译（多次使用同一正则时推荐）
pat = re.compile(r'\d+')
print(pat.findall('a1b22c333'))  # ['1', '22', '333']
```

## 七、Match 对象常用方法

| 方法 | 含义 |
|------|------|
| `group(n)` | 第 n 个捕获组内容（0 = 全部匹配） |
| `groups()` | 所有捕获组的元组 |
| `groupdict()` | 命名捕获组的字典 |
| `start(n)` | 第 n 个捕获组的起始位置 |
| `end(n)` | 第 n 个捕获组的结束位置 |
| `span(n)` | 第 n 个捕获组的 `(start, end)` 元组 |

**最小 Demo：**

```python
import re

m = re.search(r'(?P<area>\d{3})-(?P<num>\d{4})', '电话: 138-0000')

print(m.group(0))         # '138-0000'
print(m.group(1))         # '138'
print(m.groups())         # ('138', '0000')
print(m.groupdict())      # {'area': '138', 'num': '0000'}
print(m.span(1))          # (4, 7)
```

## 八、爬虫高频模式速查

| 场景 | 正则 | 说明 |
|------|------|------|
| 提取 HTML 标签内容 | `>([^<]+)<` | 取 `>` `<` 之间的文本 |
| 提取属性值 | `href="([^"]+)"` | 取双引号内的链接 |
| 提取数字 | `\d+` 或 `\d+\.?\d*` | 整数 / 小数 |
| 提取邮箱 | `[\w.-]+@[\w.-]+\.\w+` | 基本邮箱格式 |
| 提取 URL | `https?://[^\s"'<>]+` | 从文本中取链接 |
| 提取中文 | `[一-龥]+` | 连续中文字符 |
| 去除空白 | `re.sub(r'\s+', '', s)` | 清洗多余空白 |
| 非贪婪取标签 | `<tag>(.*?)</tag>` | 避免跨标签匹配 |

## 九、速查决策树

```
需要匹配位置 → ^ $ \b \B \A \Z
需要重复匹配 → * + ? {n,m}（加 ? 变非贪婪）
需要二选一   → |
需要提取子串 → ( )
需要忽略大小写 → re.I
需要 . 匹配换行 → re.S
需要多次复用 → re.compile()
需要只取不存 → (?: )
需要按名取值 → (?P<name> )
```
